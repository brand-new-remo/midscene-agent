# GitHub 交互演示
# 演示访问 GitHub 并执行多个操作 - 使用纯粹自然语言

@web:
  url: https://github.com
  headless: false
  viewportWidth: 1280
  viewportHeight: 768

@task: 搜索仓库

1. 导航到 github.com 并等待页面加载
2. 等待页面完全渲染
3. 在页面顶部的搜索框中输入 "midscene"
4. 点击搜索按钮或按回车键进行搜索
5. 等待搜索结果加载完成
6. 统计搜索结果的数量并提取前3个仓库的名称
7. 截取一张 GitHub 搜索结果的截图

@task: 查看第一个仓库

1. 找到第一个搜索结果并点击进入仓库页面
2. 等待页面加载完成
3. 截取一张仓库详情页的截图
4. 提取并显示仓库的名称、描述、主要编程语言、星标数和最近更新时间
5. 验证页面是否正确显示了仓库的详细信息
