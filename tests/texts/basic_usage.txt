# Midscene 基础使用示例
# 演示纯粹自然语言的网页自动化 - 无需使用特殊指令

@web:
  url: https://midscenejs.com/zh/introduction
  headless: false
  viewportWidth: 1280
  viewportHeight: 768

@task: 基础网页自动化示例

1. 导航到页面，等待页面完全加载完成，确认能看到页面标题和主要内容
2. 在左侧导航菜单中找到"JavaScript API 参考"菜单项，如果没有看到，可以向下滚动菜单区域找到它
3. 点击"JavaScript API 参考"菜单项，等待页面完全跳转到新页面（至少等待5秒）
4. 检查页面是否显示了"API 参考（公共部分）"标题，确认跳转成功
5. 向下滚动页面，在中间内容区域找到"agent.aiQuery()"相关内容
6. 确认在"agent.aiQuery()"区域下方能看到"返回值可以是任何合法的基本类型，比如字符串、数字、JSON、数组等。"这段文字
