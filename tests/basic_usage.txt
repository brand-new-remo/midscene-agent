# Midscene 基础使用示例
# 演示纯粹自然语言的网页自动化 - 无需使用特殊指令

@web:
  url: https://midscenejs.com/zh/introduction
  headless: false
  viewportWidth: 1280
  viewportHeight: 768
  deviceScaleFactor: 2


@task: 基础网页自动化示例


1. 在左边菜单栏区域向下滚动，找到"JavaScript API 参考"菜单项
2. 点击"JavaScript API 参考"菜单项
3. 检查页面中间区域是否显示了"API 参考（公共部分）"
4. 在页面中间区域向下滚动页面，在中间内容区域找到"agent.aiQuery()"相关内容
5. 确认在"agent.aiQuery()"区域下方能看到"返回值可以是任何合法的基本类型，比如字符串、数字、JSON、数组等。"这段文字
