# 演示如何在执行搜索后提取搜索结果 - 使用纯粹自然语言

@web:

  headless: false
  viewportWidth: 1280
  viewportHeight: 768

@task: 搜索并提取结果

1. 导航到 bing.com
2. 等待页面完全加载
3. 在页面顶部的搜索框中输入 "LangGraph DeepSeek Midscene"
4. 点击搜索按钮或按回车键进行搜索
5. 等待搜索结果加载完成
6. 截取一张搜索结果页面的截图
7. 提取并显示页面上第一个搜索结果的标题、链接和描述信息，返回 JSON 格式
8. 验证页面是否正确显示了搜索结果列表
