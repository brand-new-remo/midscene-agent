# HTTPBin 交互演示
# 演示多个连续任务的执行 - 使用纯粹自然语言

@web:
  url: https://httpbin.org
  headless: false
  viewportWidth: 1280
  viewportHeight: 768

@task: 基础交互任务

1. 导航到 httpbin.org 首页
2. 等待2秒钟让页面完全加载
3. 截取一张 HTTPBin 首页的截图
4. 查看页面上可用的端点列表
5. 提取并显示页面上显示的主要 API 端点名称和描述
6. 验证页面是否正确显示了 API 端点信息

@task: 滚动和交互测试

1. 向下滚动页面
2. 等待1秒钟让页面稳定
3. 截取一张滚动后页面的截图
4. 向上滚动回到页面顶部
5. 等待1秒钟让页面回到顶部
6. 验证页面是否已经回到顶部
